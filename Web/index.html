SugarPlayer 是一款基于 Wails3 + Vue3 打造的桌面音乐播放器，集本地音乐管理、在线搜索播放、歌词与桌面歌词于一体。

Features

为音乐而生

从本地曲库到全网音源，一个播放器全部搞定。

🎧

本地音乐管理

自动扫描本地文件夹，整理专辑、歌手与封面，支持编辑歌曲信息。

🌐

在线音乐搜索

聚合多平台音源，单曲 / 歌单 / 专辑一键搜索，畅听全网好歌。

📝

歌词与桌面歌词

精准滚动歌词，独立桌面歌词窗口，沉浸式跟唱体验。

📌

歌单固定

喜欢的在线歌单可固定到侧边栏，随时回放，无需重复搜索。

🎨

毛玻璃界面

现代亚克力质感与流畅动画，支持浅色 / 深色与窗口特效。

⚡

轻量高效

基于 Wails3 原生内核，资源占用低，启动迅速，体验顺滑。
